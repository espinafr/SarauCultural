# Sarau Cultural
Esse website foi feito com o intuito de servir como pesquisa de aprovação para uma prática extensionista da faculdade. O objetivo era, através do sarau, arrecadar alimentos para doação, mas infelizmente houveram alguns problemas técnicos e obstáculos que impediram a realização desse sarau como havíamos planejado.
O site é estático e no momento serve apenas para visualização. 

Todo o design foi planejado no figma e então passado para o código a mão através do TailwindCSS. Nesse site particularmente me desafiei a evitar ao máximo o uso de IAs, justamente para deixar minha criatividade fluir e ser mais íntimo das ferramentas que uso. Certamente me orgulho muito de como ele se desenrolou, e apesar do projeto oficial não ter ido pra frente, ainda serviu de aprendizado e rendeu bons frutos para os fins do acúmulo de conhecimento.
